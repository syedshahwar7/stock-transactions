# stock-transactions

## Install the dependencies:
`npm install`

## Build:
`npm run build`
Note: Even starting app builds the app first so this is not mandatory.

## Run the app:
`npm start [sku]`
Example: `npm start LTV719449/39/39`

## Run the tests:
`npm run test`

## Check the tests coverage:
`npm run test:coverage`
